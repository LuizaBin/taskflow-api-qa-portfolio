▶️ **How to Run Tests (Postman)**

1. Import Collection

Open Postman → Import → Select collection JSON from /Postman folder.

2. Add Environment

Import environment JSON and add:

**base_url**

**api_key**

**city**

3. Run request
   
Click Send to execute tests.

5. Run all tests
   
Use Collection Runner in Postman.

**playwright-ts-api-tests file**
