# Taskflow API QA Portfolio

This repository contains API testing artifacts for the Taskflow project.

## Contents

- **Postman Collections:** Taskflow API testing collection
- **Test Cases:** Exported from Zephyr
- **Reports:** Test execution results

## How to Use

1. Import `Postman_Collections/Taskflow_API_Testing.postman_collection.json` into Postman.
2. Configure environment variables (if any) and run the collection.
3. Refer to `Test_Cases/Taskflow_API_Test_Cases.xlsx` for detailed test cases.
